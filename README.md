# TaskPulse ⚡️

This is the main application folder for **TaskPulse**.

## Quick Start

```bash
npm install
npm run dev
```

## Features
- **Account-free**: No login required.
- **Deadline-focused**: Automated status for "At Risk" and "Overdue" tasks.
- **Privacy-first**: Local storage only.
- **Modern UI**: VS Code style collapsible layout and sticky headers.

For the full project documentation and installation guide, please refer to the [root README.md](../README.md).
